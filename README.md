## Quick Deploy to Vercel

1. Go to https://vercel.com/new → Import `noreplayreboulo/reboulo-smarthr-ai`
2. Add environment variables (see `.env.example`)
3. Deploy → Connect domain `reboulo-uae.com`

## Local Dev

```bash
npm install && cp .env.example .env.local && npm run dev
```
